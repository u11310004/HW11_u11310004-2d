地震學導論：從理論基礎到社會應用數位專題

本專題將生硬的地震學教科書基礎理論轉化為極具可讀性的數位知識系統。內容深度涵蓋地震波傳播力學、走時曲線的幾何光學推導，以及應力與應變的基礎張量關係。除了完整的理論脈絡梳理外，更進一步結合工程實務，剖析地表強震動特徵與近代減災、防災工程的實質整合應用，落實理論與社會安全的跨領域鏈結。

專家學術論壇

花蓮外海地震震源機制與地體構造應力場析論

本報告針對 2018 年 10 月 23 日發生於台灣花蓮外海的顯著有感強震展開地質構造探討。報告詳盡追蹤並比對了中央氣象署觀測網與全球即時震源機制解 (rCMT) 的反演參數。藉由下半球立體投影數據，解析出該地震在東北—西南走向上的高角度逆斷層破裂特徵，並對應至菲律賓海板塊向北隱沒至歐亞大陸板塊的 Benioff 帶幾何特徵，完整探討沖繩海槽弧後擴張帶與琉球隱沒系統交會處的複雜應力互制機制。

實地考察實務

國家地震工程研究中心（NCREE）工程耐震前沿科技體驗

親赴國家地震工程研究中心進行高階工程實地學術考察，深刻研習現代結構耐震設計的三大核心核心方針：「小震不壞、中震可修、大震不倒」。透過大型三軸地震模擬振動台、反力牆與強震即時觀測儀器的現地解說，深度探究反應譜、結構韌性配筋等核心力學概念。同時掌握了主動/被動制震器、速度型消能阻尼器及隔震支承墊的系統配置原理，有效破除大眾對於抗震工程與房屋結構防護的傳統歷史誤區。

學術生涯規劃

跨國科研視野與全球學術研究開拓座談實錄

聚焦於全球化學術研究生涯的策略性藍圖規劃。聆聽青年學者宋冠毅老師傾囊相授其中外學術研究的心路歷程，脈絡性地剖析如何從零開始發掘國際科研趨勢、撰寫具備學術說服力的研究計畫書、應對高難度國際面試，進而前往海外頂尖機構開展前沿合作。座談核心聚焦於「破除未知恐懼、主動出擊爭取學術話語權」之執行力培養，並詳盡梳理國家釋出的國際研究基金與交流管道。

互動模組開發

地球內部圈層構造與體波衰減機制動態互動網頁

本研究運用網頁前端技術，將抽象的地球內部層狀圈層構造進行視覺化重構。網頁設計的核心在於運用體波（Body Waves，包含 P 波與 S 波）的走時特徵，去解析莫氏不連續面（Moho Line）、核幔邊界（Gutenberg Line）及內核邊界（Lehmann Line）的波速跳躍。系統不僅直觀呈現了地球非均向性力學機制與震波能量衰減，更創新整合了即時自我評測答題模組，顯著強化學術推廣的互動成效。

科學數值繪圖

南東太平洋隆起規模 M 6.6 強震之震源機制反演與沙灘球繪製

本項實作精準重現了 USGS 官方事件編號 us6000sz1d（南東太平洋隆起中洋脊張裂區）的震源參數解。利用 Python 科學計算生態系（包含 Matplotlib、ObsPy 震源機制模組），透過純數值反演將複雜的矩張量數據轉換為直觀的雙力偶沙灘球（Beachball Plot）。分析證實此事件具備高達 91% 的走向滑移（Strike-Slip）分量，精確求得主斷層面與輔助共軛面的走向、傾角與滑動角，展現洋脊超高速擴張應力場的張裂運動特徵。

進階理論研析

震源矩張量力學與三維焦球立體投影理論成果展示

這份高階網頁成果報告系統性地解構了描述斷層錯動能量釋放的本質物理量——地震矩（Seismic Moment, M₀ = μDS）以及震源矩張量（Seismic Moment Tensor）的數學矩陣模型。專案藉由高質感的網頁視覺排版，細緻推導了如何將三維焦球上的初動輻射圖樣投影至二維下半球立體投影網（Stereonet）的拓撲過程，為進階地震地體構造與區域應力場分析奠定強固的理論基石。

演算法實作

虛擬下半球 P 波初動極性立體投影幾何模擬系統

本系統為一套完全自主研發的網頁端幾何模擬軟體。系統核心演算法採用下半球立體投影（Lower Hemisphere Stereonet Projection）公式，使用者可輸入全球觀測測站的專屬方位角（Azimuth）與出射角（Take-off Angle）。系統會全自動根據接收到的 P 波初動極性（壓縮 '+' 標示為紅色實心圓點；膨脹 '-' 標示為藍色空心圓圈）進行動態座標轉換與二維投影繪製，提供研究者判讀雙力偶斷層幾何面特徵的關鍵實作利器。

邊緣運算與科技藝術

【時空共鳴箱】歷史與動漫震度微縮預警儀專案開發

【時空共鳴箱】歷史與動漫震度微縮預警儀 是一款將物聯網物緣運算技術與二次元迷因流行文化完美跨界融合的創意 Raspberry Pi 地震觀測儀專案。系統核心採用樹莓派微處理器架構，透過 I2C 通訊協定以 25Hz 高頻率即時採樣 MPU6050 三軸加速度計的空間重力向量。本專案徹底打破傳統科研數據枯燥乏味的刻板印象，創新研發出「十段式時空次元震度矩陣」演算模型，賦予數據生動幽默的時代演繹。

系統三大核心科學與硬體防護機制：

  • 震度分級科學依據與演算矩陣： 系統運用歐幾里得範數（Euclidean Norm）將 MPU6050 的 X、Y、Z 三軸加速度即時融合成不限方向的合成總加速度向量（$total\_g$）。以靜止狀態的 1.0g 為基準線。當動態外力超越 1.05g 閾值時即觸發震動事件，並依據精準的振幅級距，將現實世界震度幽默投射至動漫次元：如 1.05g 對應「貓咪主子跳下床」、1.90g 聯動《火影忍者》「佩恩·神羅天征」、大於等於 2.60g 則觸發「超級賽亞人極限變身」名場面。
  • 軟體調變實體 LED 燈光防燒毀機制： 為了在微型硬體無實體限流電阻的严苛硬體配置下安全運作，專案核心引入了實體軟體 PWM（脈衝寬度調變）技術。在「即時監測模式」下，低於閾值時佔空比保持 0%（完全熄滅）；一旦超越 1.05g 判定震動，瞬間給予 15% 的安全恆亮佔空比，隨震幅維持溫和綠光。當抽到高階動漫強烈震度時，立即自動切換為「高能演習模式」，啟動 15% 與 0% 交替的數位極限爆閃（每秒 4 次），在兼顧硬體壽命的前提下，完美模擬基地警報的強烈科技張力。
  • 雲端推播連動與封包冷卻機制： 當衝擊數據達標，樹莓派邊緣端會迅速將發震時間、精密 g 力值及所對應的動漫次元場景打包成標準 JSON 封包。利用 requests 函式庫透過安全的 HTTP POST 協定即時發送至 Discord Webhook。系統內置了 10 秒的警報冷卻鎖（Cooldown Lock），能有效拦截並過濾地震波連續晃動引起的重複觸發，確保實體燈效安全並兼顧雲端 API 頻寬的防護安全性。
